class Status(event: Event? = null)

A Status represents the outcome of a function that can produces Errors and Warnings, holding any number of events.

fun addEvent(event: Event, overrideSeverity: Boolean = false)

Adds event to eventList (if it contains a custom message) and updates type if the severity of event is higher than the current severity of type. This behavior can be changed to always override the severity to the event's severity by setting overrideSeverity to true.
